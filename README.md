## 📦 Установка зависимостей

```bash
npm install
# или
yarn install
```

## 🚀 Запуск и сборка проекта

```bash
npm run dev

npm run build
```

## 🛠 Используемые технологии

```bash
React + TypeScript
React Router DOM
TanStack Query (React Query)
Axios
SCSS Modules
```
